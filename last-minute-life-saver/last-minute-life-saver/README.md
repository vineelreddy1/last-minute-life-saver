# ⚡ Last-Minute Life Saver

A hyperlocal emergency help platform connecting people with urgent needs to verified local heroes.

## 🚀 How to Run

Just double-click `index.html` — no server needed. Opens in any browser.

## 📁 Pages

| File | Description |
|------|-------------|
| `index.html` | Home / Landing page |
| `request.html` | Send SOS request |
| `helpers.html` | Browse local helpers |
| `register-helper.html` | Sign up as a hero |
| `track.html` | Live tracking page |
| `dashboard.html` | Admin dashboard |

## 🛠 Tech Stack
- Pure HTML, CSS, JavaScript (no framework needed)
- Font Awesome icons (CDN)
- localStorage for demo data persistence

## 💡 Features
- 12 emergency categories
- SOS request form with urgency levels
- Helper directory with search & filters
- Live request tracker with step-by-step progress
- Hero registration form
- Admin dashboard with charts & analytics
