## 🚨 The Problem

Every day, people face urgent crises that existing platforms are too slow to solve:

- Pipe bursts at **midnight** — no plumber answers
- Birthday cake needed **tomorrow morning** — bakeries are closed
- **Locked out** at 11pm with kids inside
- Elderly parent needs **medicine right now**

JustDial and UrbanClap require advance booking. They're not built for emergencies. Local helpers exist and want to respond — but there's no system to connect them instantly.

---

## 💡 The Solution

**Last-Minute Life Saver** is a panic button for everyday crises. Pick a category, describe your problem, set your urgency — and within seconds, verified local heroes near you receive an alert and respond.

```
User sends SOS  →  Nearby heroes alerted  →  Fastest one responds  →  Live tracking  →  Problem solved ✅
```

## 🚀 Quick Start

No installation needed. Just open in a browser:

```bash
git clone https://github.com/yourusername/last-minute-life-saver.git
cd last-minute-life-saver
open index.html          # Mac
start index.html         # Windows
xdg-open index.html      # Linux
```

## 💻 Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JavaScript
- **Storage** — localStorage (demo) → Firebase Firestore (production)
- **Notifications** — WhatsApp Business API via Twilio
- **Maps** — Google Maps API
- **Hosting** — Netlify / Vercel / GitHub Pages

Zero dependencies. Works offline. No build step required.
